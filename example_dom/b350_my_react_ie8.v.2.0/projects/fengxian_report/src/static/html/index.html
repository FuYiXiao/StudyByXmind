风险概述
针对杭州市**区*******有限公司**营业部 2017-1-1至2018-3-31的涉税数据，总局财行司、浙江省局对该纳税人进行风险分析，涉及财产和行为税司2018年第二批关于少缴城市维护建设税、教育附加和地方教育附加的风险任务，纳税人申报污染物监测数据与环保部门的监督性监测数据不一致等相关监控内容。
最终评定风险等级为中。风险指向及应对建议如下：
  1. 1、城建税及附加同增值税消费税比对中，可能少缴纳城建税及附加的风险。
  2. 2、依照税法规定，纳税人申报应税污染物排放量可按照监测机构出具的监测数据计算。但存在纳税人申报的污染物监测数据与环保部门传递的相关数据不一致的情况，致使申报的应税污染物排放量不准确，存在无法准确计算环保税额的风险。
建议尽快联系纳税人核实相关问题。
